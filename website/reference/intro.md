# 레퍼런스

:::

## 다룰 내용

| 문서 | 대상 | 상태 |
|---|---|---|
| 오픈소스 정책 (스타트업) | 개발자 10명 이하 | 준비 중 |
| 오픈소스 정책 (중소기업) | 10~100명 | 준비 중 |
| 오픈소스 정책 (대기업) | 100명 이상 | 준비 중 |
| 라이선스 검토 프로세스 | 전 규모 | 준비 중 |
| 취약점 대응 프로세스 | 전 규모 | 준비 중 |
