## 실습 전환 슬라이드 표준 형식

각 모듈 시작 슬라이드는 아래 형식을 따른다:

```
🛠 지금부터 M? 실습입니다

예상 소요시간: ?분
완료 기준: 이 파일이 생성되면 됩니다
  - output/xxx/yyy.md

실행 명령어:
  cd agents/0?-agent-name && claude
```

## 체크포인트 슬라이드 표준 형식

각 모듈 완료 슬라이드는 아래 형식을 따른다:

```
✅ M? 체크포인트

화면에 이것이 보여야 합니다:

[실제 출력 예시 텍스트]

아직 안 보인다면: 강사를 호출하거나 옆 분께 도움 요청
```
